How to develop C-DEVA plug-ins：

1,The implement of the interface

Implement the abstract class "org.java.plugin.Plugin" from JPF ,implement two methods "doStart()" and "doStop()"， Implement the interface NewAlogrithm offered by the system，different plug-in implements different methods，

Introduce the methods which used to implement a clustering algorithm：

Public String getAlgorithmNmae()：//get the name of the algorithm
Public Vector[] getClusters(String[] para)：//get the functional modules
Public String[] getParams()：//get the paramater names
Public String[] getParaValues()： //get the paramater values。
Public String getDescription()： //get the description of the algorithm
Public int getStyle()：//get the plug-in type:NewAlgorithm.Algorithm;
Public void variableInit()：//initialize the variables

Introduce the methods which used to implement a evaluation：

Public String getEvaluateNames()：//get the evaluation name；
Public void drawCharts(){String[] algorithm,HashMap<String,Vector<Node>[] resultList>,
Composite composite,HashMap<String,RGB> colorlist}：//used to evaluate the results。
Public String getDescription()： //get the description of the evaluation
Public int getStyle()：//get the plug-in type:NewAlgorithm.Evaluation;
Public void variableInit()：//initialize the variables

2,plugin.xml


To bundle a plug-in we need a .jar file,and there must be a plugin.xml file inside。
<?xml version="1.0" ?>
<!DOCTYPE plugin PUBLIC "-//JPF//Java Plug-in Manifest 1.0" "http://jpf.sourceforge.net/plugin_1_0.dtd">
<plugin id=" " version="1.0.0"
class=" ">
<runtime>
<library id="ruby" path="/" type="code"/>
</runtime>
</plugin>
This file contains the information about plug-in id,plug-in version,plug-in main class and so on

3，Example：FAG-EC plug-in

(1),Main class FAG-EC


Main class in this plug-in：
public class SecondPlugin extends Plugin implements NewAlgorithm

public int getStyle() {
return NewAlgorithm.Algorithm;
}

protected void doStart() throws Exception {
}
protected void doStop() throws Exception {
}

public String getAlgorithmName() {
return "FAG-EC";
}

public String[] getParams() {
return new String[]{"Threshold(0-1)","ComplexSize(>=2)"};
}

public String[] getParaValues() {
return new String[]{"0.1","3"};
}

public void variableInit() {
}
Public Vector<Node>[] getClusters(String[] para){};


(2),plugin.xml


<?xml version="1.0" ?>
<!DOCTYPE plugin PUBLIC "-//JPF//Java Plug-in Manifest 1.0" "http://jpf.sourceforge.net/plugin_1_0.dtd">
<plugin id="com.wuxuehong.plugin.FAG-EC" version="1.0.0"
class="com.wuxuehong.plugin.SecondPlugin">
<runtime>
<library id="FAG-EC" path="/" type="code"/>
</runtime>
</plugin>


(3),Bundle the file


Get a jar file contains the class files and plugin.xml 。
